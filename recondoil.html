Oil cleaning system

A short animation to showcase SKF RecondOil machine.

The process

For this project I decided to have a more artistic approach. Showing the dance of the dirty oil become clean thanks for SKF filter technology. It was a very fun project to make where I used liquid simulation in Unity â€“ in this way I produced in real time all the takes I needed and no long render times took place.

  • Art direction - with storyboard and visualization style planning with an intro, a message, a climax and an outro. The message and text were planned together with the marketing manager for that area. 
  • 3D modeling - using C4D and the Step files.
  • 3D animation and rendering - using Unity engine, all the animation were done inside, together with all the lighting and all takes.
  • Graphic design - text animation and fonts, together with icons as per guidelines.
  • Video editing - using Premiere, gathering all the takes. Adapting the music.
